# wf-search - 查询 Wiki

查询 Wiki 知识库，了解当前系统实现和历史决策。

## 使用方式

```bash
# 查询关键词
/wf-search "用户认证"

# 查询多个关键词
/wf-search "登录" "注册"

# 指定搜索范围
/wf-search "API" --scope current
```

## 功能说明

### 1. 搜索范围

WikiFlow 会在以下目录中搜索：

- **current/** - 当前实现（优先搜索）
- **history/** - 历史版本
- **decisions/** - 决策日志
- **deprecated/** - 已废弃功能

### 2. 搜索逻辑

- **关键词匹配**：在标题和内容中查找关键词
- **相关度排序**：根据匹配度和文档权重排序
- **上下文提取**：显示匹配内容的上下文

### 3. 结果展示

```
📊 搜索结果：用户认证

🎯 current/authentication.md (匹配度: 95%)
   用户认证系统支持邮箱登录和第三方登录...

📜 history/login-v1.md (匹配度: 78%)
   [已归档] 旧版登录系统，使用 session 认证...

💡 decisions/2024-03-15-jwt-migration.md (匹配度: 65%)
   决策：从 session 迁移到 JWT 认证...
```

## 使用场景

### 开发前了解背景

```bash
/wf-search "支付流程"
# 了解现有支付实现，避免重复开发
```

### 查询历史决策

```bash
/wf-search "为什么使用"
# 查看某个技术选型的决策原因
```

### 查找废弃功能

```bash
/wf-search "已废弃"
# 查看哪些功能已被废弃及替代方案
```

## 注意事项

- 搜索优先显示 `current/` 中的文档
- 历史版本会标注 `[已归档]`
- 决策日志可以帮助理解技术选型原因
- 搜索结果会在当前会话中缓存，避免重复查询
